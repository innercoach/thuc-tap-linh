# Tài Liệu Dự Án

Thư mục `docs/` chứa tài liệu thiết kế hệ thống và hướng dẫn đọc dự án Landing Page tuyển sinh viết bằng PHP thuần theo mô hình Custom MVC.

## Danh sách tài liệu

- [Tổng quan hệ thống](./system-overview.md)
- [Sơ đồ hệ thống](./system-diagrams.md)
- [Thiết kế cơ sở dữ liệu](./database-design.md)
- [Luồng xử lý request](./request-flow.md)
- [Hướng dẫn chạy local](./deployment-local.md)

## Mục tiêu của bộ tài liệu

- Giúp người mới hiểu nhanh cấu trúc của dự án.
- Giúp giáo viên chấm bài nắm được cách tổ chức source code.
- Làm rõ luồng xử lý từ request tới controller, model, view và database.
- Cung cấp sơ đồ hệ thống để đưa vào báo cáo hoặc thuyết trình.
